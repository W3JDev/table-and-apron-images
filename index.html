Table & Apron Menu Images

Image repository for the Table & Apron restaurant menu app

This repository hosts images for the Table & Apron menu application. All images are organized by menu category and can be accessed via direct URLs.

How to use

Images can be accessed using the following URL format:

https://w3jdev.github.io/table-and-apron-images/[category]/[image-filename]
